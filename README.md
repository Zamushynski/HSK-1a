# HSK1 汉语 — Установка на телефон

## Способ 1: GitHub Pages (бесплатно, 5 минут)

1. Создай аккаунт на [github.com](https://github.com) (если нет)
2. Создай новый репозиторий → назови его, например, `hsk1`
3. Загрузи ВСЕ файлы из этого архива в репозиторий:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icon-192.png`
   - `icon-512.png`
4. Зайди в **Settings → Pages**
5. В разделе **Source** выбери **Deploy from a branch**
6. Выбери ветку `main` и папку `/ (root)` → нажми **Save**
7. Через пару минут сайт будет доступен по адресу:
   `https://ТВОЙ_ЮЗЕРНЕЙМ.github.io/hsk1/`

## Способ 2: Netlify (ещё проще)

1. Зайди на [app.netlify.com](https://app.netlify.com)
2. Перетащи папку с файлами прямо в окно браузера
3. Готово — получишь ссылку вида `https://random-name.netlify.app`

## Установка на телефон

### iPhone (Safari):
1. Открой ссылку на сайт в **Safari**
2. Нажми кнопку **«Поделиться»** (квадрат со стрелкой внизу)
3. Выбери **«На экран Домой»**
4. Нажми **«Добавить»**

### Android (Chrome):
1. Открой ссылку в **Chrome**
2. Появится баннер **«Установить приложение»** — нажми его
3. Или: меню ⋮ → **«Добавить на главный экран»** / **«Установить»**

## Результат

Приложение появится на домашнем экране с иконкой 汉.
Работает офлайн, без полоски браузера — как обычное приложение.
